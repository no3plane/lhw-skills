# lhw-skills

我的 Skills 仓库

## 安装

```bash
# 全部安装
npx skills add no3plane/lhw-skills -g

# 指定安装
npx skills add no3plane/lhw-skills -g --skill lhw-magic14
```

## 技能

| Skill | 简介 |
|-------|------|
| [lhw-magic14](./lhw-magic14/SKILL.md) | 从十四个角度打开格局 (sleight of mouth)
| [lhw-bookfind](./lhw-bookfind/SKILL.md) | 想学某个知识领域但不知道读什么书时使用
| [lhw-book2doc](./lhw-book2doc/SKILL.md) | 把书籍、PDF、扫描版 PDF 等转换成 AI 易读的单文件文档
| [lhw-deconstruct](./lhw-deconstruct/SKILL.md) | 通通解构了
| [lhw-talk](./lhw-talk/SKILL.md) | 扩展话题
