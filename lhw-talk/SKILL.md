# lhw-talk

输入一句话，输出一张可以无限延展的对话网。

## 这是什么

不是话术。不是套路。不是"找话题"。

是把一句话拆出**三个跳转方向**——往下钻、往上跳、往旁边平移——让对话在一条线还没断的时候，就已经搭上了下一条线。线和线交织，对话就不会死。

单一脉络聊天 = 一问一答一问一答，3 分钟必尬。
多重脉络聊天 = 一句话同时埋三个钩子，每个钩子都能再分三个钩子。聊不完。

## 第一步：抓核心概念

用户给的可能是一整句话，甚至一段对话。先**圈出 1-2 个可被钻取的核心概念**——通常是动词短语或具体名词，不是形容词、不是情绪。

例子：
- 「我最近开始自己做饭了」→ 核心：**自己做饭**
- 「他下个月要去日本」→ 核心：**去日本**
- 「最近睡不好」→ 核心：**睡眠** / **最近的状态**

如果一句话里有多个候选，挑**信息密度最高、最具体**的那个。抽象词（"压力大""不容易"）不能当核心——它没法钻。

把核心概念回放给用户看：「核心抓的是 ____。从这里往三个方向打开。」

## 第二步：三个方向，每个方向 6-8 个钩子

### 方向一：子概念（往细节钻）

把核心概念**拆成更小的零件**——具体动作、工具、流程、时间、成本、感受、人物。

铁律：每个钩子必须是**可被对方接话**的具体颗粒，不是分类术语。

✅「买菜：去哪买、贵不贵、会不会挑」
❌「物质层面」——这是分类标签，对方接不下去。

### 方向二：父概念（往背景跳）

把核心概念**放进更大的背景**——生活方式、时代变化、价值观、人生阶段、社会现象。

铁律：父概念要**跳层级**，不是同义改写。

✅「自己做饭 → 一个人怎么安排生活」（跳到生活方式层）
❌「自己做饭 → 做饭这件事」（同一层换个说法）

### 方向三：兄弟概念（往同类平移）

找**和核心概念在同一层、解决类似需求、面对类似处境**的其他事。

铁律：兄弟概念是**对方也可能在做的事**，不是逻辑上的同类。

✅「自己做饭 → 点外卖 / 自己带饭 / 健身 / 早睡」（都是一个人生活的小事管理）
❌「自己做饭 → 米其林大厨」（逻辑同类，但跟对方无关）

## 第三步：每个钩子配一句"为什么这里能接"

光列钩子不够。每个钩子后面跟**一句话点出接续的角度**——是问对方的经验？是分享自己的具体细节？是抛一个反差？

格式：
```
- 钩子 —— 接续角度 / 一句话示范
```

例：
```
- 买菜 —— 同一把青菜，菜市场和超市价格能差三倍，对方多半也踩过坑
- 翻车经历 —— 自曝最惨一次，对方会笑着接自己那次
```

## 第四步：串联示范

写一段 8-12 句的对话流，把上面的钩子**真的串起来**，每句旁边标注它用了哪一类跳转。

铁律：
1. **不要走完一条线再换**——在一条线还没断的时候就跳。
2. **跳转要自然**——靠"说起来""不过""所以""倒是"这种连接词，不是硬切。
3. **最后一句必须是反问**——把球抛回对方。`你呢？` `你那边怎么样？` `你平时是 X 还是 Y？`
4. **不要在示范里说教**——不要解释"我现在在用兄弟概念跳转"。让结构自己说话，标注放在旁边括号里。

格式参考：

```
【起点：核心概念】
…………。
【子 · 具体细节】
…………，结果发现…………。
【子 · 精力成本】
不过最麻烦的是…………。
【兄弟 · 同类对照】
所以我现在特别理解为什么…………。
【父 · 价值观跳层】
不过…………这件事，其实也是…………。
【兄弟 · 抛回对方】
你平时是 X 多，还是 Y 多？
```

## 质量底线

写完之后，**逐项自检**：

1. **核心选对了吗**？核心必须可钻——抽象词、形容词、情绪词都不能当核心。
2. **每个钩子都是具体颗粒吗**？分类标签、抽象术语必须重写。
3. **父概念真的跳层了吗**？把"自己做饭"改成"做饭这件事"不算跳。
4. **兄弟概念是对方也可能在做的事吗**？不能是只有专家才碰得到的同类。
5. **串联示范里有跳转吗**？如果一条线走到底再换，就是单脉络，不及格。
6. **最后一句把球抛回去了吗**？没有反问，对话还是会死。

## 通用反例

- ❌「子概念：物质层面 / 精神层面 / 社会层面」——分类术语，不能接话
- ❌「父概念：做饭 → 烹饪文化」——只是把核心换成更书面的词，没跳层
- ❌「兄弟概念：自己做饭 → 米其林餐厅 → 分子料理」——逻辑同类，但跟对方生活无关
- ❌ 串联示范全是"我我我我我"——没有抛回对方，对话立刻死
- ❌ 用"接下来我们聊聊另一个话题"硬切——多脉络的关键是不切，而是缠绕

## 一个完整输出示范

**输入**：「我最近开始自己做饭了。」

**核心**：自己做饭

---

### 子概念 · 往细节钻

- **买菜** —— 同一把青菜不同地方差好多，对方多半踩过坑
- **菜谱来源** —— 小红书 vs B 站 vs 妈妈电话，每种都能展开
- **厨具** —— 空气炸锅、电饭煲，谁家没几个吃灰的
- **翻车经历** —— 自曝最惨一次，对方会笑着接自己那次
- **下班后的精力** —— 这是所有上班族的共同痛点
- **省不省钱** —— 直觉是省，记账后会怀疑人生
- **健康** —— 油盐是唯一能控住的变量
- **做饭时的情绪** —— 是放松还是新负担，能聊出价值观

### 父概念 · 往背景跳

- **一个人怎么安排生活** —— 从做饭跳到独立生活的整体管理
- **外卖太贵了** —— 跳到消费/经济变化
- **健康管理** —— 饮食、运动、睡眠是一套
- **从被照顾到照顾自己** —— 跳到人生阶段变化
- **年轻人的吃饭问题** —— 跳到时代/城市议题
- **工作节奏** —— 忙到没时间好好吃饭，跳到工作生活平衡
- **小时候家里怎么吃饭** —— 跳到原生家庭/记忆
- **给谁做饭** —— 跳到关系层面（朋友、伴侣、家人）

### 兄弟概念 · 往同类平移

- **点外卖** —— 最直接的对照
- **自己带饭** —— 同一个动作的另一种形态
- **下馆子** —— 解决吃饭的另一条路
- **咖啡 / 奶茶** —— 同样是日常小消费
- **健身** —— 一个人生活的另一项自我管理
- **收拾房间** —— 同样是"自己照顾自己"
- **记账** —— 顺手就能聊到的相邻习惯
- **早睡** —— 自我管理小事联盟

### 串联示范

```
【起点：自己做饭】
我最近开始自己做饭了。

【子 · 买菜】
一开始只是觉得外卖吃腻了，结果发现买菜也挺有学问的——
同样一把青菜，菜市场和超市差好多。

【子 · 精力成本】
但最麻烦的不是买菜，是下班之后真的没精力洗菜切菜。

【兄弟 · 外卖对照】
所以我现在特别理解为什么大家宁愿点外卖。

【父 · 健康管理】
不过自己做饭确实会健康一点，至少油盐能控制住。

【兄弟 · 记账】
我最近甚至开始顺手记账，想看看自己做饭到底有没有比外卖省。

【父 · 独立生活】
说起来这其实不只是做饭——一个人生活以后，
吃饭、睡觉、收拾房间、花钱，全得自己管。

【兄弟 · 抛回对方】
你平时是自己做饭多，还是外卖多？
```
